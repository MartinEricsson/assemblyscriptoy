How to write AssemblyScript for Gasm

1. Export a main function

Export a function named main(). The Gasm compiler will automatically wrap it with a @compute shader entry point.

export function main(): void {
  // your code here
}

2. Memory Access

Use load<T>(offset) and store<T>(offset, value) to interact with the single memory buffer shared with the GPU.

const count = load<i32>(0);
store<i32>(4, count * 2);

3. Supported Types

Currently, Gasm supports i32, u32, f32, and scalar math operations. i64 and f64 are demoted to 32-bit equivalents.

4. Advanced Memory Layout

The first elements of memory can be used for passing uniforms like frame count or configuration flags.
