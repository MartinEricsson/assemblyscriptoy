> 01 · EXPORT A MAIN FUNCTION

Export a function named main(). The Gasm compiler wraps it in a @compute shader entry point.

export function main(): void {
  // your code here
}

> 02 · MEMORY ACCESS

Use load<T>(offset) and store<T>(offset, value) to interact with the single memory buffer shared with the GPU.

const count = load<i32>(0);
store<i32>(4, count * 2);

> 03 · SUPPORTED TYPES

Gasm supports i32, u32, f32, and scalar math. i64 and f64 are demoted to 32-bit equivalents.

> 04 · ADVANCED MEMORY LAYOUT

The first elements of memory can be used for passing uniforms like frame count or configuration flags.
